# setup

```sh
git clone git@github.com:Dj-Viking/aoc-2021.git; \
cd aoc-2021/CSharp; \
```

# development
```sh
# build C# dll files
sh build.sh
```

# run
```sh
# to solve a particular day and part, place the numbers as arguments to the shell script
# usage: sh solve.sh <day number> <file name>
sh solve.sh 1 input
```
